## Overview

This file implements MCPBridge, an adapter that turns a RuntimeHost into discoverable MCP tools (getTools) and a single execution entrypoint (handleToolCall). getTools returns tool descriptors (name, description, inputSchema) for orchestrators to discover available operations. handleToolCall validates incoming arguments, normalizes values, invokes the appropriate RuntimeHost method (spawnAgentFromTemplate, spawnSwarmFromTemplate, signalAgent, orchestrateSwarm, listAgentTemplates, getRuntimeSnapshot, listEnvironmentIds), and returns results as a single text block containing pretty-printed JSON. Validation helpers enforce required/optional types and enums; errors are raised as thrown Error instances. The bridge favors human-readable JSON text output to simplify integration with LLM-driven tooling.

## Architecture Notes

- Bridge pattern: MCPBridge adapts MCP tool calls to RuntimeHost method invocations.
- Asynchronous integration: handleToolCall is async and awaits runtime methods while validators are synchronous.
- Validation-first design: strict input validation and normalization before runtime invocation.
- Error handling: validation and unknown tools surface as thrown Errors.
- Response shape: responses are returned as a single pretty-printed JSON text block.

## Usage Examples

### Spawn a single agent from a template

Call getTools() to inspect the spawn_agent schema, then call handleToolCall('spawn_agent', { agentId, environmentId, type, config? }). The bridge validates inputs, calls runtime.spawnAgentFromTemplate, and returns a text block with JSON describing the result.

### Spawn a swarm of agents

Call handleToolCall('spawn_swarm', { type, swarmId, environmentId, size, roles?, continueOnError? }). Validation enforces size is a positive integer and roles is an array of non-empty strings before invoking runtime.spawnSwarmFromTemplate.

### Signal an agent (PAUSE/RESUME/TERMINATE/SNAPSHOT)

Call handleToolCall('signal_agent', { agentId, signal }). The bridge validates the signal against the allowed enum and calls runtime.signalAgent(agentId, signal), returning a confirmation JSON in a text block.
