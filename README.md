## Quick Start

### Prerequisites
- Node.js 20+
- pnpm 8+
- Redis
- Docker (optional)

### Installation

```bash
# Clone the repo
git clone https://github.com/omnigents/omnigents.git
cd omnigents

# Install dependencies
pnpm install

# Copy environment template
cp .env.example .env

# Start development
pnpm dev
```

### Docker Development

```bash
# Start all services
docker-compose -f docker-compose.dev.yml up -d

# View logs
docker-compose logs -f

# Stop services
docker-compose down
```
